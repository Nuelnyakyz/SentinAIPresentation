Intelligent System Monitoring

Next-generation observability powered by AI. Detect anomalies, analyze behavior, and remediate issues in real-time.

Core Modules

📊

MetricsPulse

Utilizes Prometheus for data collection and trains a RandomForest model for prediction. It raises an alert when current CPU/RAM usage exceeds predictions by over 50%.

📝

LogsPulse

Aggregates logs using Loki and trains a machine learning model to classify them live. It instantly categorizes system events to detect errors and security threats.

👤

UserBehaviorPulse

Connects via SSH and uses psutil to monitor remote user activity. It builds a whitelist of normal behavior to flag any suspicious or anomalous commands.

Support Architecture

Dashboard

Built with Streamlit, this interactive interface aggregates real-time streams from all Pulse modules. It provides a unified view for monitoring system health and reviewing historical data.

Intelligence

The cognitive engine of SentinAI. It leverages Large Language Models (LLMs) and XAI to interpret complex anomaly patterns, offering natural language explanations and precise remediation steps.

Notifications

Integrates with the Telegram Bot API for instant alert delivery. It pushes high-severity alerts and AI-generated summaries to administrators' mobile devices in real-time.

System In Action

Real-time Metrics & Graphs

Visualize system performance with dynamic line charts that predict future load. The system learns your infrastructure's heartbeat to distinguish between normal spikes and genuine anomalies.

  • Predictive Modeling: Uses historic data to forecast expected CPU and RAM usage.
  • Threshold Validation: Immediately flags deviations exceeding 50% of the predicted baseline.
  • Live Updates: Charts refresh automatically, giving you a pulse on system health.

Log Distribution Analytics

Gain global visibility into your system's operational footprint. Interactive pie charts and distribution graphs provide immediate context on what's happening across your entire infrastructure.

  • Severity Ratios: Quickly assess the proportion of High, Medium, and Low severity events.
  • Error Categorization: Automaticaly groups logs by source (e.g., Auth, Database) to identify hotspots.
  • Volume Tracking: Visualizes log density to help you spot sudden spikes that often precede failures.

Live Logs Feed

Monitor system events as they happen with sub-second latency. The feed offers dynamic filtering by severity, source, and category, ensuring you can isolate critical errors from the noise instantly.

Features include:

  • Real-time auto-refresh mechanism.
  • Color-coded severity tags (Low, Medium, High).
  • One-click filtering for rapid triage.

Behavior Analysis Table

Track user actions against established baselines. This module provides a granular view of command execution to detect insider threats or compromised accounts.

  • Process Monitoring: Logs every executed command and its associated user.
  • Anomaly Detection: Flags "unexpected" commands that deviate from a user's whitelist profile.
  • Severity Grading: Automatically assigns risk levels (Safe, Warning, High) to every action.

Granular Log Explanations

Don't just read logs; understand them. Select any individual entry to trigger an on-demand AI analysis. The system breaks down:

  • Root Cause: Why the event occurred based on process context.
  • Impact Analysis: Potential risks to stability or security.
  • Remediation: Concrete, actionable steps to resolve the issue.
  • Confidence Score: AI's certainty level in its assessment.

Overall System AI Explanation

SentinAI continuously synthesizes data from all modules to provide a holistic health report. Instead of checking ten different charts, get a single "General Status" summary.

The AI Engine evaluates:

  • Current Metrics vs. Predictions: flagging deviations in CPU or RAM usage.
  • Log Severity Counts: Aggregating high/medium alerts.
  • Behavioral Anomalies: summarizing unusual user activity.
  • Actionable Findings: providing a "Normal", "Warning", or "Crucial" status with clear reasoning.

Interactive AI Assistant

Your expert system companion. Engage in a natural language dialogue with SentinAI to ask questions like "How is the server performing?" or "Explain the recent CPU spike."

Capabilities include:

  • Instant Snapshots: Returns a real-time table of metrics and predictions.
  • Contextual Awareness: Remembers previous queries for follow-up questions.
  • Deep Dives: Can query effective log patterns and explain complex technical states in simple English.
  • 24/7 Availability: Always ready to assist with troubleshooting and system audits.

Instant Notifications

Stay informed anywhere. The system automatically pushes notifications to the SentinAI-bot on Telegram whenever a warning or high-severity alert is triggered.

  • Immediate Delivery: Alerts reach your phone seconds after detection.
  • Rich Context: Notifications include the error summary, severity level, and specific host details.
  • Actionable insights: AI-generated summaries accompany alerts, letting you understand the "why" without logging in.
